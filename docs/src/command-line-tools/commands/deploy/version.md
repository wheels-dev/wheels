# `wheels deploy version`

Print the deploy toolchain's version and upstream Kamal alignment.

## Synopsis

```bash
wheels deploy version
```

## Description

Prints a one-line version string indicating the upstream Kamal version and `kamal-proxy` version this release mirrors. Use to confirm parity when reporting bugs.

## Examples

```bash
$ wheels deploy version
wheels-deploy mirrors kamal 2.4.0 / kamal-proxy v0.8.6
```
