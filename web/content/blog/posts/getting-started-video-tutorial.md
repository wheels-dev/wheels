_\[Update: A newer version of this has been released as a "Hello World" tutorial. We've updated with that with a link to the new video rather than the old one.\]_ Here is the first Wheels video tutorial. To be honest it's the first screen cast I've created also, but I think the quality turned out pretty good. This tutorial covers how to download and install the latest Wheels build, create a couple of pages, add dynamic content to the pages, link the pages together, and finally modify the default framework settings so one of the new pages is displayed as the default page.
