Along with the [0.8 release of Wheels](http://www.cfwheels.com/blog/index.cfm/2008/8/18/Released-ColdFuson-on-Wheels-Version-08) came a whole slew of new chapters in our [Documentation](http://www.cfwheels.com/docs). If you were looking into learning a little bit more about the framework over the weekend, the [Docs](http://www.cfwheels.com/docs) are a great place to start. For those of you who have been following along since before 0.8, here's what's new:

-   [Frameworks and Wheels](http://www.cfwheels.com/docs/chapter/frameworks-and-wheels)
-   [Obfuscating URLs](http://www.cfwheels.com/docs/chapter/obfuscating-urls)
-   [Event Handlers](http://www.cfwheels.com/docs/chapter/event-handlers)
-   [Column Statistics](http://www.cfwheels.com/docs/chapter/column-statistics)
-   [Dynamic Finders](http://www.cfwheels.com/docs/chapter/dynamic-finders)
-   [Getting Paginated Data](http://www.cfwheels.com/docs/chapter/getting-paginated-data)
-   [Displaying Links for Pagination](http://www.cfwheels.com/docs/chapter/displaying-links-for-pagination)
-   [Switching Environments](http://www.cfwheels.com/docs/chapter/switching-environments)

All other chapters are updated for 0.8 as well. As for the future of documentation, here are some of my hopes and dreams:

-   Documentation at the function level for the Wheels API (which is done for the most part... I just need to get it formatted and on the site!)
-   A better navigation system for the Docs themselves
-   A "Livedocs" format akin to [how Adobe does it](http://livedocs.adobe.com/coldfusion/8/htmldocs/index.html)

Please [let us know](http://groups.google.com/group/cfwheels) if we're not clear about something in the Docs or if you have any ideas!
