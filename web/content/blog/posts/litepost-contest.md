ColdFusion On Wheels is announcing a contest to create an example of [LitePost](http://en.wikipedia.org/wiki/LitePost_%28blogging_application%29) using Wheels The LitePost project is hosted [here](http://code.google.com/p/litepost/). It is a simple blogging application to compare different ColdFusion frameworks. A sql script for the tables and other framework examples are [available](http://litepost.googlecode.com/svn/trunk/).

### Goal

Fun exercise to learn Wheels for newbies and chance to win an Amazon Gift card.

### Prize

Grand Prize (one): ~$50~ _$100_ Amazon Gift card Runner Up Prizes (two): ~$25~ _$50_ Amazon Gift card Everyone: Free one year Mini Hosting Plan at [Alurium Hosting](http://alurium.com/) (over $40 value)

### Rule

Only one rule for the contest - create a LitePost example using Wheels philosophies such as [conventions over configuration](https://cfwheels.org/docs/chapter/conventions), no xml, [Object Relational Mapping](https://cfwheels.org/docs/chapter/object-relational-mapping), [Object Validation](https://cfwheels.org/docs/chapter/object-validation), etc. No [ColdSpring](http://www.coldspringframework.org/) requirement for component management.

### Requirements

- [Download](https://cfwheels.org/download) and use the Wheels framework
- Login/Logout
- Entries (Create, Read, Update, and Delete)
- Comments (Create, Read, Update, and Delete)
- Categories (Create, Read, Update, and Delete)
- Bookmarks (Create, Read, Update, and Delete)
- Use Wheels Object Validation on above items
- RSS feed (Bonus)

Contest ends on **February 19, 2010**. Winners will be picked by me (Mike Henke). I will be taking bribes and will be partial to newbies :-)

### Submissions

Please email entries to henke.mike@gmail.com include your name and your Wheels/LitePost example. Any code emailed may be used by ColdFuson On Wheels.

### Getting Started

If you are not familiar with Wheels, review the two tutorials, [Beginner Tutorial: Hello World](https://cfwheels.org/docs/chapter/beginner-tutorial-hello-world) and [Beginner Tutorial: Hello Database](https://cfwheels.org/docs/chapter/beginner-tutorial-hello-database). The [screencasts](https://cfwheels.org/screencasts) are a great way to quickly get started. If you have more time, user group recordings are [available](http://www.carehart.org/ugtv/list.cfm?search=wheel). Direct any questions to the [Wheels Google Group](http://groups.google.com/group/cfwheels)
