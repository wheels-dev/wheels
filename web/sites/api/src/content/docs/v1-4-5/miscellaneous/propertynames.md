## Signature

`propertyNames()` — returns `any`




## Description

Returns a list of property names ordered by their respective column's ordinal position in the database table. Also includes calculated property names that will be generated by the Wheels ORM.


## Examples

<pre>propertyNames() &lt;!--- Get a list of the property names in use in the user model ---&gt;
&lt;cfset propNames = model(&quot;user&quot;).propertyNames()&gt;</pre>
