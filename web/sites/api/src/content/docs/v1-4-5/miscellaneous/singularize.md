## Signature

`singularize()` — returns `any`




## Description

Returns the singular form of the passed in word.

## Parameters

<div class="wd-params-table">

| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `word` | `string` | yes | — | String to singularize. |

</div>

## Examples

<pre>singularize(word) &lt;!--- Singularize a word, will result in &quot;language&quot; ---&gt;
#singularize(&quot;languages&quot;)#</pre>
