## Signature

`titleize()` — returns `any`




## Description

Capitalizes all words in the text to create a nicer looking title.

## Parameters

| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `word` | `string` | yes | — | The text to turn into a title. |

## Examples

<pre>#titleize(&quot;Wheels is a framework for ColdFusion&quot;)#
-&gt; CFWheels Is A Framework For ColdFusion</pre>
