## Signature

`columns()` — returns `any`




## Description

Returns an array of columns names for the table associated with this class. Does not include calculated properties that will be generated by the Wheels ORM.


## Examples

<pre>columns() &lt;!--- Get the columns names in the order they are in the database ---&gt;
&lt;cfset employee = model(&quot;employee&quot;).columns()&gt;</pre>
