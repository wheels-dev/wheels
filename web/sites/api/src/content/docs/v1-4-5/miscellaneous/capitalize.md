## Signature

`capitalize()` — returns `any`




## Description

Returns the text with the first character converted to uppercase.

## Parameters

<div class="wd-params-table">

| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `text` | `string` | yes | — | Text to capitalize. |

</div>

## Examples

<pre>capitalize(text) &lt;!--- Capitalize a sentence, will result in &quot;Wheels is a framework&quot; ---&gt;
#capitalize(&quot;wheels is a framework&quot;)#</pre>
