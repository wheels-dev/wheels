## Signature

`singularize()` — returns `string`

**Available in:** `controller`, `model`, `mapper`, `migrator`
**Category:** String Functions

## Description

Returns the singular form of the passed in word.



## Parameters

| Name | Type | Required | Default | Description |
| ---- | ---- | -------- | ------- | ----------- |
| `word` | `string` | yes | — |  |

## Examples

<pre>// Singularize a word, will result in &quot;language&quot; 
#singularize(&quot;languages&quot;)#</pre>
