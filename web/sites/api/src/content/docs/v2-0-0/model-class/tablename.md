## Signature

`tableName()` — returns `string`

**Available in:** `model`
**Category:** Miscellaneous Functions

## Description

Returns the name of the database table that this model is mapped to.




## Examples

<pre>// Check what table the user model uses
whatAmIMappedTo = model(&quot;user&quot;).tableName();</pre>
