Stop and remove `kamal-proxy` on every host.

<Aside type="danger">
All inbound traffic stops. Re-run [`proxy boot`](/v4-0-1-snapshot/command-line-tools/commands/deploy/proxy/boot/) or [`wheels deploy setup`](/v4-0-1-snapshot/command-line-tools/commands/deploy/setup/) to restore.
</Aside>

## Synopsis

``` title="synopsis"
wheels deploy proxy remove [--destination=<name>] [--dry-run]
```

## Flags

| Flag | Description |
|------|-------------|
| `--destination=<name>` | Overlay destination config. |
| `--dry-run` | Print commands without executing. |

## Example

```bash title="illustrative — requires remote hosts"
wheels deploy proxy remove
```
