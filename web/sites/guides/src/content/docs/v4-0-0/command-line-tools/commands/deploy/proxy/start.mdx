`docker start kamal-proxy` on every host. Fails if the container doesn't exist — use [`proxy boot`](/v4-0-0/command-line-tools/commands/deploy/proxy/boot/) for first-time install.

## Synopsis

``` title="synopsis"
wheels deploy proxy start [--destination=<name>] [--dry-run]
```

## Flags

| Flag | Description |
|------|-------------|
| `--destination=<name>` | Overlay destination config. |
| `--dry-run` | Print the commands without executing. |

## Example

```bash title="example"
wheels deploy proxy start
```
