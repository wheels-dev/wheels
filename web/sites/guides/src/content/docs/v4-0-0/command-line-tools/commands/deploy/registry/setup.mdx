Alias for [`registry login`](/v4-0-0/command-line-tools/commands/deploy/registry/login/). Kept for parity with Kamal's setup verb.

## Synopsis

``` title="synopsis — illustrative"
wheels deploy registry setup [--password=<value>] [--destination=<name>] [--dry-run]
```

Behavior identical to [`registry login`](/v4-0-0/command-line-tools/commands/deploy/registry/login/).

## Example

```bash title="illustrative"
wheels deploy registry setup
```
