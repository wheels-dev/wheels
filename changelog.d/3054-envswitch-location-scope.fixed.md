- URL environment switching to `production` or `maintenance` (`?reload=<environment>&password=...`) no longer 500s and silently reverts to the file-configured environment. `vendor/wheels/events/onapplicationstart.cfc` made a bare `$location()` call on the `redirectAfterReload=true` + `url.reload` present branch (auto-enabled for production/maintenance), but that file is a bare `component {` with no helper scope, so Lucee threw `No matching function [$LOCATION] found` and the application cold start aborted. The call now goes through `application.wo.$location(...)` like the other 27 helper calls in the file.
