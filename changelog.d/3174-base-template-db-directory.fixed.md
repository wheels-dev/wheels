- Build: the published `wheels-base-template` zip omitted the conventional `db/` directory because `tools/build/base/.gitignore` carried a `/db/**` rule and CommandBox `package publish` honors `.gitignore`, silently stripping `db/.keep` — so SQLite/H2 file paths under `db/` failed without a manual `mkdir`. The ignore rule now whitelists `!/db/.keep` so the directory ships while end-user db files stay ignored, pinned by a new structural guard spec (`BaseTemplateDbDirectoryShipsSpec`) (#3174)
