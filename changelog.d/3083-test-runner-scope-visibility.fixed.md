- The `/wheels/core/tests` and built-in app-test (`app-runner.cfm`) JSON runners now surface scope resolution instead of silently reporting green for the wrong scope: the payload carries `directoryRequested`, `directoryResolved`, `directoryRejected`, `bundlesDiscovered`, and a `warnings[]` array, so a `directory=` value that fails the allowlist (silently swapped for the full default suite) and an allowlist-passing single-spec-file value that discovers 0 bundles are both detectable from the response rather than masquerading as a passing run (#3083)
