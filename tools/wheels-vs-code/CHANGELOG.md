# Change Log

All notable changes to the "wheels-vs-code" extension will be documented in this file.

Check [Keep a Changelog](http://keepachangelog.com/) for recommendations on how to structure this file.

## [1.0.0] - 2025-09-03

### Added
- Initial release with 308 Wheels code snippets
- Complete coverage of framework API functions including:
  - Controller functions (actions, filters, rendering, flash, pagination)
  - Model functions (CRUD, associations, validations, callbacks)
  - Migration functions (table creation, columns, indexes, foreign keys)  
  - View helpers (forms, links, assets, sanitization)
  - Configuration functions (settings, routing, environments)
  - Testing functions (TestBox integration)
- Parameter placeholders with tab navigation for all snippets
- Support for Adobe ColdFusion 2018+, Lucee 5.3+, and BoxLang 1.0+
- Professional extension icon and branding
